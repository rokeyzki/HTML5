# 事件属性
HTML 元素可拥有事件属性，这些属性在浏览器中触发行为
***

## onabort 属性 ※
> ### 说明：
* 发生 abort 事件时执行脚本

> ### 有效值：
* script

## onbeforeonload 属性 ※
> ### 说明：
* 在元素加载前执行脚本

> ### 有效值：
* script

## onblur 属性
> ### 说明：
* 当元素失去焦点时执行脚本

> ### 有效值：
* script

## onchange 属性
> ### 说明：
* 当元素改变时执行脚本

> ### 有效值：
* script

## onclick 属性
> ### 说明：
* 在鼠标点击时执行脚本

> ### 有效值：
* script

## oncontextmenu 属性 ※
> ### 说明：
* 当菜单被触发时执行脚本

> ### 有效值：
* script

## ondblclick 属性
> ### 说明：
* 当鼠标双击时执行脚本

> ### 有效值：
* script

## ondrag 属性 ※
> ### 说明：
* 只要脚本在被拖动就执行脚本

> ### 有效值：
* script

## ondragend 属性 ※
> ### 说明：
* 在拖动操作结束时执行脚本

> ### 有效值：
* script

## ondragenter 属性 ※
> ### 说明：
* 当元素被拖动到一个合法的放置目标时执行脚本

> ### 有效值：
* script

## ondragleave 属性 ※
> ### 说明：
* 当元素离开合法的放置目标时执行脚本

> ### 有效值：
* script

## ondragover 属性 ※
> ### 说明：
* 只要元素正在合法的放置目标上拖动时，就执行脚本

> ### 有效值：
* script

## ondragstart 属性 ※
> ### 说明：
* 在拖动操作开始时执行脚本

> ### 有效值：
* script

## ondrop 属性 ※
> ### 说明：
* 当元素正在被拖动时执行脚本

> ### 有效值：
* script

## onerror 属性 ※
> ### 说明：
* 当元素加载的过程中出现错误时执行脚本

> ### 有效值：
* script

## onfocus 属性
> ### 说明：
* 当元素获得焦点时执行脚本

> ### 有效值：
* script

## onkeydown 属性
> ### 说明：
* 当按钮按下时执行脚本

> ### 有效值：
* script

## onkeypress 属性
> ### 说明：
* 当按键被按下时执行脚本

> ### 有效值：
* script

## onkeyup 属性
> ### 说明：
* 当按钮松开时执行脚本

> ### 有效值：
* script

## onload 属性
> ### 说明：
* 当文档加载时执行脚本

> ### 有效值：
* script

## onmessage 属性 ※
> ### 说明：
* 当 message 事件触发时执行脚本

> ### 有效值：
* script

## onmousedown 属性
> ### 说明：
* 当鼠标按钮按下时执行脚本

> ### 有效值：
* script

## onmousemove 属性
> ### 说明：
* 当鼠标指针移动时执行脚本

> ### 有效值：
* script

## onmouseover 属性
> ### 说明：
* 当鼠标指针移动到一个元素上时执行脚本

> ### 有效值：
* script

## onmouseout 属性
> ### 说明：
* 当鼠标指针移出元素时执行脚本

> ### 有效值：
* script

## onmouseup 属性
> ### 说明：
* 当鼠标按钮松开时执行脚本

> ### 有效值：
* script

## onmousewheel 属性 ※
> ### 说明：
* 当鼠标滚轮滚动时执行脚本

> ### 有效值：
* script

## onresize 属性 ※
> ### 说明：
* 当元素调整大小时执行脚本

> ### 有效值：
* script

## onscroll 属性 ※
> ### 说明：
* 当元素滚动条被滚动时执行脚本

> ### 有效值：
* script

## onselect 属性
> ### 说明：
* 当元素被选中时执行脚本

> ### 有效值：
* script

> ### 说明：
* 当表单提交时执行脚本

> ### 有效值：
* script

> ### 说明：
* 当文档卸载时执行脚本

* script
